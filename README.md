## Getting Started

To run this project locally:

1. Clone the repo:
   ```bash
   git clone https://github.com/yourusername/task-manager-api.git
   
2. Navigate into the project folder:
    ```bash
   cd task-manager-practice
   
3. Run the app:
    ```bash
   mvn spring-boot:run
